# Dotfiles

My personalized config files for unix systems

## Requirements

  - GNU Stow
  - Vim

## How to install?

    $ git clone -r https://github.com/marcos-ro/dotfiles ~/.dotfiles
    $ cd .dotfiles
    $ git submodule update --init --recursive
    $ stow home
